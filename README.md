# 🛒 Local Store Shopping Website

A modern and responsive **Local Store Shopping Website** built using **HTML, CSS, and JavaScript**.
This project allows users to browse products, search items, add products to cart, and place orders through a smooth shopping experience.

## 🌐 Live Demo

🔗 **Website:** `https://localshopstore.netlify.app`

Visit the live website here:
https://localshopstore.netlify.app

## ✨ Features

* 🛍️ Product listing with categories
* 🔍 Product search functionality
* 🛒 Add to cart system
* ➕ Increase / decrease product quantity
* ❌ Remove products from cart
* 📦 Order summary and checkout
* 💬 WhatsApp integration
* 🚚 Local delivery support
* 📱 Fully responsive design
* 🎨 Modern UI/UX with smooth animations

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript**
* **Local Storage** (for cart management)

## 📂 Project Purpose

This project was built to practice and improve my **frontend web development skills** by creating a real-world **local shopping website** with cart functionality and a responsive user interface.
